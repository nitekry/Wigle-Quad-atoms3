# Wigle-Quad-atoms3
Allows you to use Atoms3 screen to indicate by color if you are outside of you WWWD quad.
Libraries  
M5AtomS3	- M5Stack  
M5Unified	- M5Stack  
TinyGPS++	- ArduPilot/TinyGPSPlus
